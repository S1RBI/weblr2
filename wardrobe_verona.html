Шкаф распашной "Верона" (4-х дверный)

Описание товара

Классический четырехдверный распашной шкаф "Верона" с элегантными фасадами. Вместительное решение для хранения одежды и белья в спальне или гардеробной.

Характеристики товара

  1. Размеры (ШхВхГ), мм: 1600 х 2100 х 520
  2. Материал корпуса: ЛДСП (цвет Белый, Дуб Сонома)
  3. Материал фасада: МДФ с фрезеровкой (цвет Белый глянец, Крем)
  4. Количество дверей: 4 (четыре)
  5. Петли: Стандартные, с возможностью установки доводчиков (опция)
  6. Внутреннее наполнение: Две секции со штангой для одежды, одна секция с полками
  7. Ручки: Металлические, классический дизайн

Подробное описание товара

Шкаф "Верона" привнесет в ваш интерьер нотки классической элегантности. Фасады из МДФ с изящной фрезеровкой выглядят стильно и дорого. Продуманное внутреннее наполнение позволяет удобно разместить большое количество вещей. Распашные двери обеспечивают полный доступ к содержимому шкафа.

Преимущества модели:

  • Элегантный классический дизайн.
  • Качественные фасады из МДФ с фрезеровкой.
  • Большая вместительность.
  • Удобное зонирование внутреннего пространства.
  • Надежная фурнитура.

Материалы и уход

Корпус шкафа изготовлен из ЛДСП толщиной 16 мм. Фасады - МДФ. Для ухода за поверхностями используйте мягкую влажную ткань. Глянцевые фасады рекомендуется протирать специальными средствами для глянцевых поверхностей, чтобы избежать разводов.
